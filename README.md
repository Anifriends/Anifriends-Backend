## 🔎 프로젝트 소개

`봉사자`는 도움이 필요한 보호소를 한 번에 모아서 보고, `보호소`는 봉사자의 봉사 이력과 봉사 온도를 통해 신뢰할 수 있는 봉사자의 도움을 받을 수 있는 프로젝트입니다.
